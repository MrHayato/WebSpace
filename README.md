#To install:

~~~ { .bash }
git clone https://MrHayato@github.com/MrHayato/WebSpace.git
npm install
~~~

#To run:

~~~ { .bash }
node web.js
~~~

##As a Server:
  Open your browser to localhost:8899 and create a server

##As a Client:
  Open your browser to localhost:8899/join and join a server
